Сборщик
Требуется сборка
<&multiblock>Сборщик - это сложная машина, способная создавать предметы, подобно тому, как это делал бы игрок.
Чтобы сформировать структуру, постройте её, как показано выше, и нажмите ПКМ Молотом по одному из конвейеров.<np>
Конвейер, на который вы нажмёте, будет входной стороной, обозначенной синим маркером и Стрелкой на стороне машины.
Сборщик позволяет хранить в себе до 3 рецептов, которые будут выполняться по порядку. Выходные данные предыдущих рецептов могут быть использованы в качестве входных данных для следующих рецептов.
3 поля для крафта хранят рецепты, а 18 слотов под ними - это инвентарь для материалов.<np>
На входной стороне, помимо порта для предметов, имеется также порт для жидкостей, необходимый для заполнения трёх внутренних резервуаров машины.
Жидкость в этих резервуарах может использоваться для замены ёмкостей с жидкостью, таких как вёдра с водой в рецептах.
Энергия для машины подаётся через верхний центральный блок.
