Автоматический Верстак
Для тех, кто не умеет управлять палкой
<&multiblock>Автоматический верстак - один из первых крупных успехов мелкосерийного машиностроения.
Машина может следовать набору инструкций, описанных <link;blueprints;чертежом инженера;start> для детальной сборки предметов.<np>
После сборки мультиблока на его чертежный стол можно положить чертежную схему. После выбора элемента, который должен быть создан,начнется процесс сборки. Для того чтобы машина работала, она должна быть снабжена материалами и энергией.
Входы элементов - это два люка, отмеченные синими точками, вход питания находится над ними. Готовые изделия сбрасываются в конце конвейерной ленты.
Сигнал редстоуна под таблицей чертежей можно использовать для управления.
