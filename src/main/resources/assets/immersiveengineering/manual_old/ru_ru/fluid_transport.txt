Транспортировка жидкости
Профессиональный сантехник
<&pipe_recipe>Транспортировка жидкостей стала очень важной в машиностроительном бизнесе. Жидкостные трубы и насосы помогут именно в этом. Сами по себе Жидкостные трубы перекачивают жидкость довольно медленно, но если для подачи жидкости используется насос, жидкость будет двигаться намного быстрее.<np>
Вы можете щелкнуть ПКМ по соединительному элементу трубы с помощью Молота инженера, чтобы предотвратить его соединение с этой стороной.
Жидкостные трубы могут быть покрыты деревянными или стальными лесами. Просто щёлкните ПКМ блоком строительных лесов по трубе, чтобы закрыть её. Сломайте трубу, чтобы забрать строительные леса.<np>
<&pump_recipe>Жидкостные насосы выполняют три основные функции. Их можно использовать для установки в трубы с высоким давлением, чтобы перекачивать жидкости намного быстрее, или они могут забирать жидкости из мира.
Ударяйте молотом инженера по основанию насоса, чтобы переключать стороны между входом и выходом.<np>
Для того, чтобы насос мог собирать блоки жидкости, ему потребуется подвод энергии сверху и сигнал редстоуна. Насосы также можно использовать для извлечения жидкостей из резервуаров и других ёмкостей, которые не имеют автоматической подачи. Для этого также потребуется сигнал редстоуна, но не потребуется энергия.<np>
Блоки воды будут служить бесконечным источником жидкости для насоса, если есть более 3-х блоков.

Собираемые блоки жидкости будут заменяться булыжником, чтобы уменьшить нагрузку от потоков жидкости. Это поведение можно включать и выключать с помощью удара Молота инженера по насосу.<np>
<&outlet_recipe>Выходной шлюз для жидкостей действует как конечная точка для системы трубопроводов, позволяя разливаться жидкость в мире. Стороны могут быть настроены между входом, выходом (открытым шлюзом) или заблокированы вовсе, сигнал редстоуна может использоваться для блокировки выхода. Жидкость будет извлекаться только до высоты своего уровня.
