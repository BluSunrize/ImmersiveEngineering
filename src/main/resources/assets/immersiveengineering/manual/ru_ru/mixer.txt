Миксер
Чаны с тестом для печенья!
Миксер представляет собой многоблочную структуру, которая растворяет твердые вещестра в жидкостях, смешивая их вместе. Структура строится, как показано выше, и сформирована с помощью Молота инженера на центральном блоке из листового металла.
Чаще всего эта машина используется для смешивания жидкого бетона из воды, 2 частей песка, 1 части гравия и 1 части глины или создания зелий из воды и их различных ингредиентов.
Жидкости вводятся в смеситель через порт в нижней части источника питания. Элементы можно вводить через два люка сзади. Порт для жидкости на передней панели служит выходом для жидкости.
Применение сигнала красного камня к панели управления на передней панели остановит работу машины, это поведение можно изменить с помощью Молота инженера.
