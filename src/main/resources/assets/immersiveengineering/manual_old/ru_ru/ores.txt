<&alu>§lАлюмиевая руда§r находится в мире между слоями <config;i;ores.bauxite.min_y> и <config;i;ores.bauxite.max_y>, и для добычи требуется каменная кирка.
Из руды будут плавиться алюминиевые слитки, которые вместе со сталью используются для создания высоковольтных проводов.
<&lead>§lСвинцовая руда§r находится в мире между слоями <config;i;ores.lead.min_y> и <config;i;ores.lead.max_y>, и для добычи требуется железная кирка.
Из руды будут плавиться свинцовые слитки, которые используются для создания конденсаторов.
<&silver>§lСеребряная руда§r находится в мире между слоями <config;i;ores.silver.min_y> и <config;i;ores.silver.max_y>, и для добычи требуется железная кирка.
Из руды будут плавиться серебряные слитки, которые смешиваются с золотом, чтобы создать электрум, гораздо более эффективный проводник.
<&nickel>§lНикелевая руда§r находится в мире между слоями <config;i;ores.nickel.min_y> и <config;i;ores.nickel.max_y>, и для добычи требуется железная кирка.
Из руды будут плавиться никелевые слитки, которые смешиваются с медью, чтобы создать константан, термостойкий проводник, используемый при создании <link;thermoelectric;Термоэлектрического генератора;start>.
<&uranium>§lУрановая руда§r находится в мире между слоями <config;i;ores.uranium.min_y> и <config;i;ores.uranium.max_y>, и для добычи требуется железная кирка.
Из руды выплавляются урановые слитки, но в настоящее время у этого металла не так много применений. Урановый блок может служить источником тепла для <link;thermoelectric;Термоэлектрического генератора;start>, а металл будет использоваться в будущих начинаниях.
