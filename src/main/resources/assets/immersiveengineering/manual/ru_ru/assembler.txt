Монтажник
Требуется монтажная сборка
<&multiblock>Монтажник-это сложная машина, способная создавать предметы, подобно тому, как это сделал бы игрок.
Чтобы сформировать конструкцию, постройте ее, как показано выше, и используйте молоток на одном из конвейеров.<np>
Конвейер, на который вы нажмете, будет входной стороной, обозначенной синим маркером над ней и Стрелкой на стороне машины.
Ассемблер позволяет хранить в нем до 3 рецептов, которые будут выполняться по порядку. Выходные данные предыдущих рецептов могут быть использованы в качестве входных данных для следующих рецептов.
3 сетки для крафта хранят рецепты, а 18 слотов под ними-инвентарь для материалов.<np>
Входная сторона, в дополнение к входу элемента, также имеет вход жидкости для заполнения трех внутренних резервуаров машины.
Жидкость в этих резервуарах может быть использована для замены жидких контейнеров, таких как ведра для воды в рецептах.
Питание для машины подается через верхний центральный блок.
