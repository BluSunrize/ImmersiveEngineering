Ферментер
Скал!
Ферментер представляет собой многоблочную структуру, которая извлекает этанол из определенных фруктов, зерен и овощей в процессе ферментации. Структура построена, как показано выше, и сформирована с помощью Молота инженера на центральном котле со стороны, где находятся инженерные блоки.
Предметы можно вводить в ферментер через два люка на задней панели, отмеченные синими точками. Люк с оранжевой точкой служит для вывода предметов, а блок под ним - для вывода жидкостей.
Применение сигнала красного камня к панели управления на передней панели остановит работу машины, это поведение можно изменить с помощью Молота инженера.
