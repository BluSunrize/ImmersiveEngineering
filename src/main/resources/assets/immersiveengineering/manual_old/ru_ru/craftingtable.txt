Рабочий стол инженера
Для различных проектов
Рабочий стол инженера - это простое улучшение обычного верстака. Помимо полированной, обработанной деревянной поверхности, он удерживает ингредиенты внутри поля для крафта, когда интерфейс закрыт.
Он также оснащен небольшим ящиком для хранения некоторых компонентов для крафта.
