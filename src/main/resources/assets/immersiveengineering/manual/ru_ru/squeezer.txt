Промышленная маслобойня
Актуальные вопросы
Промышленная маслобойня представляет собой многоблочную структуру, которая выжимает сок или масло из органических материалов с помощью большого поршня. Конструкция построена, как показано выше, и формируется с помощью использования Молота инженера на центральной деревянной бочке со стороны, на которой расположены инженерные блоки.
Предметы можно вводить в Промышленную маслобойню через два люка сзади, отмеченные синими точками. Люк с оранжевой точкой служит для вывода предметов, а блок под ним - для вывода жидкостей.
Применение сигнала красного камня к панели управления на передней панели остановит работу машины, это поведение можно изменить с помощью Молота инженера.<np>
