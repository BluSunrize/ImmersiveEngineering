Транспортировка жидкости
Профессиональный сантехник
<&pipe_recipe>Транспортировка жидкостей стала очень важной в машиностроительном бизнесе. Жидкостные трубы и жидкостные насосы помогут именно в этом. Сами по себе Жидкостные трубы перекачивают жидкость довольно медленно, но если для подачи жидкости используется насос, жидкость будет двигаться намного быстрее.<np>
Вы можете щелкнуть ПКМ по соединительному элементу трубы с помощью Инженерного молота, чтобы предотвратить его соединение с этой стороной.
Трубы для жидкости могут быть покрыты деревянными или стальными лесами. Просто щелкните ПКМ куском строительных лесов по трубе, чтобы закрыть ее. Сломайте трубу, чтобы забрать строительные леса.<np>
<&pump_recipe>Жидкостные насосы выполняют три основные функции. Их можно использовать для вставки в трубы с высоким давлением, чтобы перекачивать жидкости намного быстрее, или они могут забирать жидкости из мира.
Использование молотка по основанию насоса будет переключать стороны между входом и выходом.<np>
Для того, чтобы насос мог собирать блоки жидкости, ему потребуется подвод энергии сверху и сигнал красного камня. Насосы также можно использовать для извлечения из резервуаров и других емкостей с жидкостью, которые не имеют автоматической подачи. Для этого также потребуется сигнал красного камня, но не потребуется энергия.<np>
Водоблоки будут служить бесконечным источником жидкости для насоса, если есть более 3-х блоков.

Собраные блоки жидкости будут заменены булыжником, чтобы уменьшить нагрузку от потоков жидкости. Это поведение можно включать и выключать с помощью молотка инженера на насосе.<np>
<&outlet_recipe>Выходное отверстие для жидкости действует как конечная точка для системы трубопроводов, позволяя размещать жидкость в мире. Стороны могут быть настроены между входом, выходом (открытые ворота) или закрытыми, сигнал красного камня может использоваться для выключения выхода. Блок будет выводить жидкость только до своего собственного уровня высоты.
