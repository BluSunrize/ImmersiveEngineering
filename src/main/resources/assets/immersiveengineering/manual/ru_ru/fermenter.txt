Бродильный аппарат
Skál!
Бродильный аппарат - многоблочная структура, которая извлекает этанол из определённых фруктов, зёрен и овощей в процессе брожения. Строится, как показано выше, и формируется Молотом инженера по центральному котлу со стороны, где находятся инженерные блоки.
Предметы можно подавать через два люка на задней панели, отмеченные синими точками. Люк с оранжевой точкой служит для извлечения предметов, а блок под ним - для извлечения жидкостей.
Подача сигнала редстоуна на переднюю панель - остановит работу машины, это поведение можно изменить с помощью Отвёртки инженера.
