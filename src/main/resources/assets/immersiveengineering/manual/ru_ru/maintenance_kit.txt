Комплект для обслуживания
Ремонт на ходу
<&recipe>Среди тех инженеров, которые путешествуют по миру или проводят большую часть времени в шахтах, одна проблема образуется сама собой. Их инструменты требуют постоянного обслуживания, а носить с собой верстак просто нецелесообразно. К счастью, большинство мелких ремонтов и изменений конфигурации можно выполнить с помощью комплекта для обслуживания.

Набор инструментов, завернутый в прочную ткань из конопли, для обслуживания позволяет вам настраивать ваши инструменты так же, как это делает <link;workbench;Верстак инженера>. Вам больше не придется заниматься шлифовкой поверхности, чтобы сменить головку горного сверла, или возвращаться в мастерскую, чтобы настроить защитные наушники. Все это теперь можно выполнять на ходу с помощью набора для обслуживания!
