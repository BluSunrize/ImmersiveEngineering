Автоматизированный верстак
Для тех, кто не умеет ездить на механике
<&multiblock>Автоматизированный верстак - один из первых крупных успехов мелкосерийного машиностроения.
Машина может следовать набору инструкций, описанных <link;blueprints;чертежом инженера;start> для детальной сборки предметов.<np>
После сборки мультиблока на его чертёжный стол можно положить чертёж. После выбора предмета, который должен быть создан, начнётся процесс сборки. Для того чтобы машина работала, она должна быть снабжена материалами и энергией.
Входы для предмета - это два люка, отмеченные синими точками, вход для энергии находится над ними. Готовые изделия сбрасываются в конце конвейерной ленты.
Для управления можно использовать сигнал редстоуна под чертёжным столом.
